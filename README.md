## Features

- Authentication
- Reset Password
- Upload Profile
- Change Password
- Change Email
- Teacher mode and Student Mode
- Watch Video
- Create Course
- Update Course
- Delete Course
- Search Course
- Payment



## Tech Stack

**Client:** React, Redux, TailwindCSS

**Server:** NestJs, MySQL

**Deployment:** Railway, AWS, Vercel
